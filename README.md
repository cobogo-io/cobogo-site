## 1. Install packages

```
yarn
or
npm install
```

## 2. Run Locally:

```
yarn dev
```

## 3. Build:

```
yarn build
```

## 4. Run from build:

```
yarn start
```

## 5. Deploy to vercel:

```
https://nextjs.org/docs/deployment
```
